# Divii

## Intro
Divii is an app that where you can make a prediction, set a reminder  timeline and then when that time line expires, you are promted verify if that predication came true or not.
"Divii" is derived from "Divination" a class taught in the Harry Potter School of Hogwarts where young wizards learn the art/skill of predicting the future.

- Analogies 'TripleByte' for investors type professions.
- Journal that helps check your economic predictions.
- Behace for investors.

## MVP Feature Set
Initially the main screen will be divided into 3 portions. 
- Past Predictions
- Today's predictions
- Upcoming predictions

Each pediction will be a short paragraph. You can feel free to add links to _why_ you  made that prediction. Once a prediction is set, you cannot edit it. You can ammend extra information, but the prediction itself is permenant to you.

When a "future prediction's" due date arrives, you are promted to verify if it came true or not. Again, you can include links to determine if it was true or not.
